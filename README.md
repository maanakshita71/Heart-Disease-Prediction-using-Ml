#  Heart Disease Prediction System using Machine Learning

## 📌 Project Overview
This project is a **Machine Learning-based Heart Disease Prediction System** developed using Python, Flask, and Scikit-learn.  
It predicts whether a person is at risk of heart disease based on medical parameters such as age, blood pressure, cholesterol, ECG results, and more.

The project includes a **professional web dashboard** where users can input health details and get real-time predictions.

---

## 🎯 Objective
To build an intelligent system that helps in early detection of heart disease using machine learning techniques, improving preventive healthcare analysis.

---

## 🧠 Machine Learning Model
- Algorithm used: **Random Forest Classifier**
- Preprocessing: **OneHotEncoding + Pipeline**
- Dataset: **Heart Disease UCI Dataset (Kaggle)**
- Output: Binary classification (Heart Disease Risk / No Risk)

---

## 📊 Features Used in Model
- Age 
- Chest Pain Type  
- Resting Blood Pressure  
- Cholesterol  
- Fasting Blood Sugar  
- Rest ECG  
- Maximum Heart Rate  
- Exercise Induced Angina  
- Oldpeak (ST depression)  
- Slope of ST segment  
- Number of Major Vessels (CA)  
- Thalassemia (Thal)
